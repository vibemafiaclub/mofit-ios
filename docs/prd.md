# PRD: 모핏 (Mofit) — AI 트레이너 iOS 앱

## 목표
전면 카메라로 스쿼트를 실시간 추적하고 개수를 세주는 iOS 앱. AI 코칭으로 운동 피드백 제공.
MVP를 빠르게 출시하여 시장 반응을 검증한다 (YC 지원 대상).

## 핵심 가치
- **운동 트래킹이 핵심**. AI 코칭은 부가 가치.
- 속도 우선. 최소 화면, 최소 기능, 외부 의존성 0.
- 서버 없음. 로그인 없음. 로컬 전용.

## 사용자
운동 습관을 만들고 싶은 개인. 별도 장비 없이 폰 하나로 운동을 추적하고 싶은 사람.

## 기능 범위

### 온보딩 (필수, 최초 1회)
- 단계별 입력: 성별 → 키(cm) → 몸무게(kg) → 체형 → 목표
- 체형: 마른 / 보통 / 근육질 / 통통한
- 목표: 체중 감량 / 근력 증가 / 체형 개선
- 뒤로가기로 이전 단계 수정 가능
- 입력값 검증:
  - 키: 허용 100~250, 경고 100~140·200~250, 거부 범위 밖
  - 몸무게: 허용 20~300, 경고 20~35·150~300, 거부 범위 밖

### 홈탭
- "모핏" 타이틀 (좌상단), 프로필 편집 버튼 (우상단)
- 운동 종류 선택 영역 (탭 → 바텀시트 그리드)
- 운동 시작 버튼 (형광초록, 크게)
- 오늘의 기록 요약 (세트, rep, 시간). 기록 없으면 "첫 운동을 시작해보세요!"
- 운동 종료 후 복귀 시 폭죽 효과

### 운동 선택 (바텀시트)
- 2열 그리드: 스쿼트 / 푸쉬업 / 런지 / 플랭크
- MVP에서는 전부 내부적으로 스쿼트로 처리

### 트래킹 화면
- 전체화면 카메라 프리뷰 + 오버레이
- 상태 머신: 대기 → 손바닥 1초 → 카운트다운 5초 → 운동 추적 → 손바닥 1초 → 세트 완료 + 카운트다운 5초 → 다음 세트 → ... → 종료
- 오버레이: 세트 번호(좌상), 경과 시간(우상), rep 수(중앙, 크게), 종료 버튼(하단 중앙, 빨간 원)
- 운동 추적 중 관절 포인트 화면에 표시
- 종료 → 기록 저장 → 홈 이동
- 운동 시간 = 첫 카운트다운 시작 ~ 종료 버튼

### 프로필 수정 (풀스크린 모달)
- 온보딩 5개 값 전부 수정 가능
- 저장 버튼

### 기록탭
- 상단: 가로 스크롤 날짜바 (최근 7일, 좌 스와이프로 이전 날짜)
- 선택 날짜의 세션 리스트 (운동 종류, 세트, rep, 시간, 시작 시각)
- 스와이프 삭제 가능
- 기록 없는 날: "이 날은 운동 기록이 없어요"

### AI 코칭탭
- 운동 전 / 운동 후 피드백 버튼 (사용자가 직접 선택)
- 하루 2회 (전 1, 후 1), 자정 리셋
- 사용한 버튼 비활성화, 남은 횟수 표시
- 피드백은 카드 형태로 쌓임, 최신만 펼쳐져 있음
- 기록 0일 때: "아직 운동 기록이 없어서 피드백을 드리기 어려워요"
- API 실패 시 횟수 차감 안 함

## 디자인
- 다크모드 고정
- 무채색 + 포인트 색상: 형광초록
- 탭바 3개: 홈 / 기록 / AI코칭 (SF Symbols, 선택 시 형광초록)

## 인증 및 데이터 저장

### 인증
- 이메일 + 비밀번호 기반 로그인/회원가입
- Apple 연동 없음, 비밀번호 찾기 없음
- 회원가입 완료 시 해당 정보로 즉시 로그인 상태 전환 (자동 로그인)

### AI 코칭 탭 접근 제어
- 비로그인 시 "로그인 후 사용할 수 있습니다" 안내 화면 표시
- 로그인/회원가입 버튼 제공

### 데이터 저장 분기
- 로그인 유저: 서버 API를 통해 Supabase DB에 데이터 저장
- 비로그인 유저: 기존 SwiftData에 로컬 저장
- 로컬 → 서버 데이터 마이그레이션 없음

### 로그아웃
- ProfileEditView 하단에 로그아웃 버튼 배치
- 로그인 상태에서만 표시

### Claude API 프록시
- 기존 앱 내 직접 Claude API 호출을 서버 경유로 전환
- 로그인 유저만 사용 가능

## MVP 제외 사항
- 카메라 미인식 안내, 세트 완료 진동, 폰 위치 가이드
- 앱 강제종료 시 운동 데이터 복구
- SwiftData 마이그레이션
