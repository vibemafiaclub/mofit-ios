# Mofit — 폰 하나로 자세가 교정되는 AI 스쿼트 트레이너

## 1. 가치제안 요약 (랜딩페이지 카피)

### 메인 카피

**"폰 세워두고 손바닥 1초. 스쿼트가 자동으로 세어집니다."**

iPhone 카메라만으로 스쿼트를 자동으로 세고, 관절 각도 기반으로 실시간 자세를 보여줍니다. 별도 장비도, 회원가입도 필요 없습니다. 앱 켠 지 20초 안에 첫 카운트가 올라갑니다.

### 서브 카피

- **자동 카운팅 (스쿼트 전용, MVP 단계)**: hip-knee-ankle 각도 판정으로 폼이 맞을 때만 1회로 인정. 수기 체크 필요 없음.
- **로그인 없이 바로 시작**: 설치 → 30초 온보딩 → 즉시 카운트. 모든 기록은 내 폰 로컬(SwiftData)에만 저장.
- **AI 코칭은 선택 (로그인 시)**: 내 키/몸무게/목표 + 최근 7일 운동 데이터 기반 피드백. 서버는 데이터 프록시 역할만, 카메라 영상은 어디에도 저장되지 않음.
- **광고 없음 · 결제 없음**: MVP 단계는 전면 무료. 기본 카운팅·기록·자세 오버레이는 영구 무료로 유지 계획.

### 타깃 시나리오

> "헬스장 3개월 등록했는데 2주 못 갔어요. PT는 회당 7~10만원, 유튜브 홈트는 내 폼이 맞는지 모르겠어요."

Mofit은 바로 이 지점을 겨냥합니다. 폰을 벽에 기대 세우고, 손바닥을 카메라에 1초 보여주면 5초 카운트다운 후 추적이 시작됩니다. 스쿼트가 끝날 때까지 폰을 만지지 않아도 됩니다.

### 현재 지원 범위 (솔직한 고지)

- **스쿼트만 실제 판정 동작함.** 푸쉬업/싯업 버튼은 "준비중" 배지 + 회색 톤으로 표시되며, 탭하면 "현재는 스쿼트만 지원합니다" 토스트만 나오고 트래킹으로 진입하지 않습니다.
- 스쿼트 외 운동의 실제 판정 로직은 검증 후 순차 추가 예정. **지금 설치하면 스쿼트 전용 앱으로 쓰게 됩니다.**

## 2. 구체적 기능 설명 (미팅 / 데모 전달 시)

### 2.1 온보딩 (최초 1회, 약 30초)

성별 → 키 → 몸무게 → 체형(마른/보통/근육질/통통한) → 목표(체중 감량/근력 증가/체형 개선). 회원가입·이메일 요구 없음. 입력은 전부 폰 로컬(SwiftData)에 저장.

### 2.2 홈 탭

- 오늘의 기록 요약 (세트 수, 총 rep, 시간)
- "운동 시작" 형광초록 버튼
- 운동 종류 선택 (ExercisePicker):
  - **스쿼트**: 풀컬러, 탭 시 즉시 트래킹 진입
  - **푸쉬업 / 싯업**: "준비중" 배지 + opacity 0.4 회색 톤. 탭 시 토스트 "현재는 스쿼트만 지원합니다" 1.5초 노출, 트래킹 진입 차단.

### 2.3 트래킹 화면 (카메라)

상태 머신:

```
대기 → 손바닥 1초 → 카운트다운 5초 → 운동 추적
     → 손바닥 1초 → 세트 완료 → 카운트다운 5초 → 다음 세트 → ... → 종료 버튼
```

- **스쿼트 판정**: Apple Vision Framework의 `VNDetectHumanBodyPoseRequest`로 hip-knee-ankle 각도 분석. 각도 > 160°일 때 서있음, < 100°일 때 앉음. 서있음 → 앉음 → 서있음 = 1 rep.
- **손바닥 판정**: `VNDetectHumanHandPoseRequest`로 5개 손가락이 1초 펴진 상태 감지 → 세트 시작/종료 신호.
- 관절 포인트가 화면에 오버레이로 표시되어 자세 확인 가능.
- 프리뷰 30fps, 분석 15fps 샘플링 (배터리 절약 + 스쿼트 속도에 충분).

### 2.4 기록 탭

- 상단: 최근 7일 가로 스크롤 날짜바
- 선택 날짜의 세션 리스트 (운동 종류, 세트 수, 총 rep, 시간, 시작 시각)
- 스와이프 삭제

### 2.5 AI 코칭 탭 (로그인 시에만 활성)

- 하루 2회 호출: 운동 전 동기부여 / 운동 후 피드백 (자정 리셋)
- Claude Sonnet 4.6 모델, **앱 → 자체 서버 `/coaching/request` → Anthropic API** 경유 (API 키는 앱 바이너리에 없음)
- 전달 컨텍스트: 사용자 프로필 + 최근 7일 요약(운동일수, 총 세션, 총 rep, 일평균 rep) + 일별 추이(rep/세트/세트당 평균)
- API 실패 시 일일 횟수 차감 안 함
- 비로그인 상태에서 이 탭 진입 시 AuthGateView가 뜨고 로그인/가입 유도

### 2.6 인증 (선택)

- 이메일 + 비밀번호 가입/로그인. Supabase DB + 자체 bcrypt·JWT (ADR-010).
- JWT는 Keychain에 저장, 앱 재실행 시 자동 로그인.
- **로그인 유저**: 프로필/운동 기록/AI 피드백이 서버(Supabase)에 저장.
- **비로그인 유저**: 모든 데이터 로컬(SwiftData)에만 저장, 외부 전송 0.
- **로컬 → 서버 마이그레이션 없음** (ADR-013). 로그인 여부는 의사결정 시점부터의 기록만 분기.

### 2.7 카메라 권한 / 프라이버시

- 카메라 영상은 실시간 분석 후 폐기. 저장/전송 없음.
- 서버가 받는 건 관절 좌표가 아니라 "rep 횟수 · 시간" 같은 집계 수치만 (로그인 유저 기록 저장 경로에 한함).

## 3. 가격 / 플랜

### 단계 0 (현재, MVP)

**완전 무료. 광고 없음. 회원가입 없음.** 기본 기능(스쿼트 카운팅·기록·자세 오버레이)은 영구 무료로 유지할 계획.

### 단계 1 (가설, 수개월 후)

- 기본 카운팅·기록·자세 오버레이: 영구 무료 유지
- 프리미엄 (가설): AI 코칭 호출 횟수 상향 + 주간 리포트 + 운동별 폼 점수 상세

단계 1 BM은 사용자 수·리텐션 검증 후 확정. 지금 설치하는 사용자에게는 "완전 무료 MVP 단계"를 확약.

### 단계 2 (가설, 장기)

B2B — 헬스장/PT 숍용 고객 운동 데이터 대시보드. 개인 사용자에게는 영향 없음.

## 5. 리스크 (솔직한 고지)

- **현재 스쿼트만 실제 판정.** 푸쉬업/싯업은 "준비중"으로 명시 표기. 실제 판정 로직은 단계적 추가.
- **rep 카운트 정확도**: Apple Vision 기반이라 조명·각도·옷차림에 따라 오차 가능. **권장 환경**: 발끝에서 약 2m 거리, 허리 높이, 측면 촬영. 내부 테스트에서는 권장 환경 기준 100회 연속 오차 수 회 수준이지만, 아직 공개 가능한 정량 수치는 확보 전.
- **iOS 17+ 전용** (iPhone XS 이상). Android·iPad 미지원.
- **앱 스토어 출시 전 단계**로 리뷰·별점·다운로드 수 등 공개 지표는 아직 없음.

## 6. 즉시 시작하는 방법

1. App Store에서 "Mofit" 검색 (출시 예정)
2. 앱 실행 → 온보딩 30초 → 홈 → "스쿼트" 선택 → 운동 시작
3. 폰을 벽에 기대 세우기 → 화면 안에 서서 손바닥 1초 펴기
4. 5초 후 카운트 시작. 끝나면 손바닥 1초 더 → 세트 완료.

설정도, 로그인도, 결제도 없습니다. 카메라 권한만 허용하면 끝. AI 코칭을 받고 싶을 때만 선택적으로 로그인하면 됩니다.
